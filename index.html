Active Defense

Kestrel correlates eBPF kernel telemetry with threat intelligence to detect and respond to adversarial behavior in real time. Define detection campaigns using YAML rules, chain weighted consensus events, and trigger precise countermeasures.

Kestrel comes pre-configured to leverage the MITRE ATT&CK® ineresting signals.

Detection Rule Format
detections:
  ssh_authorized_keys_manipulation:
    threshold: 0.70
    window: "90s"
    context_key: uid
    cooldown: "300s"
    tactics:
      TA0003: { name: "Persistence", base_severity: high }
    technique:
      id: "T1098.004"
      confidence: 0.90
    events:
      TA0003_T1098_004_DC0039:
        program: BPF_PROG_TYPE_LSM
        weight:  0.70
        filters:
          - ctx_present == 1
          - paths contains [ /root/.ssh/, /home/ ]
          - exe not contains [ sshd, ssh-copy-id, ansible, puppet ]
      TA0003_T1098_004_DC0061:
        program: BPF_PROG_TYPE_KPROBE
        weight:  0.50
        filters:
          - paths == [ /home/ ]
          - exe not contains [ sshd, ssh-copy-id, ansible, puppet ]
      TA0003_T1098_004_DC0032:
        program: BPF_PROG_TYPE_TRACEPOINT
        weight:  0.35
        filters:
          - comm contains [ sshd, bash, sh, ansible ]
          - exe not contains [ sshd, ssh-keygen, ssh-copy-id ]
    actions:
      - log
      - notify
      - isolation
